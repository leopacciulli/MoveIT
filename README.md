## 🤖 Front-End
The Front-End was developed with Next.js / React.js / TypeScript.

<br>

## 🏁 Starting the project
Clone: `git clone https://github.com/leopacciulli/MoveIT.git`

<br>

🤖 Running
<br>Open the terminal and run the following commands:

````zsh
$ cd MoveIT

$ yarn

$ yarn start
````
